# Techno Funda Screener

Techno Funda Screener is an AI-powered stock screening tool inspired by Mark Minervini's techno-funda methodology. It helps retail investors identify high-potential stocks using a combination of relative strength, earnings growth, and technical breakout patterns.

## Features
- Screen stocks based on Minervini criteria (RS, EPS, Sales, Breakout)
- Pull live data using Yahoo Finance
- Streamlit interface for easy exploration

## Upcoming

- Portfolio insights
